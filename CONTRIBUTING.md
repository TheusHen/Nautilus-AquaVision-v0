# Contributing

Contributions are welcome, especially:

- reproducible benchmarks on independently collected images;
- false-positive and false-negative examples that contributors have the right to share;
- data-pipeline tests and format support;
- deployment benchmarks on edge hardware;
- documentation and model-card improvements.

Do not submit copyrighted dataset images without redistribution permission, private camera footage, personal data, secrets, or credentials.

By contributing code, you agree that it is distributed under AGPL-3.0-or-later.
